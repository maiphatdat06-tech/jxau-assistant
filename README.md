# JXAU-Assistant | 江农校园助手 

基于 **Vue 3** + **uni-app** 开发的响应式校园服务小程序。

## 项目亮点
- **响应式 GPA 计算**：封装了一套符合校内标准的绩点自动计算算法，支持动态课程增删。
- **跨端适配优化**：利用 `rpx` 与 `Flex` 布局，并针对校庆 120 周年 Banner 采用 `aspectFit` 模式解决拉伸问题。
- **兼容性 Debug**：利用 Vue 的 {{ }} 语法，强制让程序把 > 当作字符串处理，彻底解决跨端环境下的箭头符号乱码问题。
- **现代前端流**：深度实践 Vue 3 组合式 API (`setup`) 与 ES6+ 语法。

## 技术栈
- **框架**：Vue 3.x, uni-app
- **交互**：uni-app 原生 API (showModal / showToast)
- **工程化**：Git 版本管理, HBuilderX
